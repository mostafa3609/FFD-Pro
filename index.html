A Unified Layered Deformation Engine

FFD PRO is an advanced, C++ native modifier for Autodesk 3ds Max. It completely revolutionizes standard Free-Form Deformation by introducing a non-destructive, multi-layered approach, a built-in Sculpting Engine, and 25 mathematically precise parametric deformers wrapped into a single, high-performance modifier.

OpenMP Multi-threading

Natively parallelized to smoothly deform heavy meshes (500k+ vertices) with zero lag.

Primary Use Cases

Parametric Modeling Build complex shapes mathematically (Spirals, Accordions, Waves) without destructive topology edits.
Procedural Animation Every parameter, Gizmo, Center, and Morph weight is animatable. Perfect for looping effects and motion graphics.
Mesh Modification Live sculpting brushes, geometry conforming, and targeted vertex manipulation (Push/Relax/Circleify).

MaxScript vs C++ — What Changed?

FFD PRO started as a free MaxScript experiment. The C++ version is a complete rewrite from scratch — faster, richer, and built to last.

1. Lattice & Layers (Resolution)

Manage the fundamental grid size and organize your deformation stack.

Lattice Resolution

Set the physical point count of your control cage via the X, Y, and Z spinners.

Lattice Resolution
  • Update All: Rebuilds the entire lattice structure for all layers.
  • Update Layer: Rebuilds only the currently selected layer. Under Development (Coming in next version)

Layer Stack Management

Add, remove, reorder, and rename your modifier stack inside FFD PRO.

Layer Stack

Easily combine multiple algorithms by stacking deformers and moving them up or down the evaluation hierarchy.

3. Layer Properties

Lattice Type & Masking

Masking & Lattice Types
  • New Lattice: Starts a fresh grid.
  • Last Lattice: Inherits previous deformations for smooth blending.
  • Solid Box: Pure mathematical mesh deformation.
  • Enable Masking: Select CPs, click Set to bake a mask. Only those points will be affected by this layer's deformer.

Axis, Limits & Morph Snapshots

  • Define the operation axis (X, Y, Z) and use Flip to reverse the math vector.
  • Limit Effect: Bound the physical range of the deformer using Base/Top limits.
  • Morph Snapshots: Convert manual sculpts/CP edits into a morph target driven by a Weight slider, freeing up points for new edits.

4. Selection & CP Tools

Smart Selection

Selection Tools
  • All X/Y/Z: Selects complete planes based on current selection.
  • Loop: Selects a continuous ring based on two seed points.
  • Grow/Shrink: Expands or shrinks selection smartly.

Topology Align

Planarize & Circleify
  • Planarize X/Y/Z: Flattens points perfectly along an axis plane.
  • Circleify: Averages selected points into a perfect circle based on their local bounding box.

Conform to Shape

Conform to Shape

Projects Control Points directly onto the surface of the underlying mesh. Use filters for Inside/Outside points and adjust Offset distance.

Animate All

Animate All

Automatically assigns Point3 controllers to all Control Points, enabling instant Keyframe recording for manual point animation.

5. Gizmo & Center Transform

Gizmo Transform

Every layer possesses its own independent, fully animatable Gizmo and Center point. You can move them freely in the viewport or use precision spinners.

Reset Tools

Reset Tools

Easily revert transforms to zero. Reset CPs returns points to base shape, Reset Volume resets the Gizmo box, and Reset Center centers the pivot.

6. Deform & Sculpting Engine

Volume & Interpolation Splines

Volume & Interpolation

Only In Volume vs All Vertices

Only In Volume limits deformation strictly to geometry inside the lattice. All Vertices affects the whole mesh regardless of lattice bounds.

Falloff Spinner

When 'All Vertices' is ON, adjusting the Falloff smoothly fades the deformation effect outwards into the rest of the geometry outside the lattice box.

Tension Spinner

Controls the tightness of the TCB Spline mathematical curves. Lower values = looser smooth curves. Higher values = rigid, straight boxy lines between points.

Continuity Spinner

Modifies the tangent angles of the grid. Tweaking this creates sharp, angular corners or overshooting loops in the interpolation.

Live Sculpting

Sculpting Brushes

Check Enable Sculpt Mode to draw directly on the viewport. A visual brush ring will appear on the lattice points.

  • Brushes: Push/Pull, Move/Grab, Pinch, Relax/Smooth, Scale, Rotate.
  • Controls: Adjust Radius and Strength. Use X/Y/Z checkboxes for live mirror Symmetry.

Hotkeys:

• Left Click + Drag: Sculpt

• ALT + Drag: Invert Brush (e.g. Pull instead of Push)

• SHIFT + Drag: Temporarily activate Relax/Smooth brush

• Right-Click: Exit Sculpt mode

7. Display & Settings

Display Overlays

Display Overlays
  • Lattice: Show/hide yellow/orange deformation cage.
  • Source: Show original undeformed green reference grid.
  • See-Thru: Draws lattice on top of geometry (Ignores Z-Buffer).
  • CPs: Show/hide Control Point markers.

Macro Listener Output

Macro Listener

Toggles whether interactions echo commands to the MaxScript Listener. Disable these on highly dense meshes to prevent UI lag.

Available toggles: CPs, Gizmo, Spinners, Reset.
